# Step 002: Karpathy Coding Laws

Implementing behavioral heuristics based on Andrej Karpathy's observations to eliminate "Vibe Coding" patterns.

## The Why
Agents tend to hallucinate or over-complicate when operating in a vacuum. These laws create a "Feedforward Harness" that restricts agent behavior to senior engineering standards.

## Technical Concept
- **Feedforward Control:** Constraining the agent's action space *before* it executes.
- **Surgical Precision:** Ensuring the diff noise is minimized to maintain project legibility.

## The Lesson
Teach the agent to "think out loud" about its assumptions. This allows the human architect to correct the world model before a single line of code is written.
