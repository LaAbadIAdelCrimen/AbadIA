# Monastic Horarium

The **Horarium** is the primary "Temporal Harness" of the game. It dictates the agent's expected location and behavior at all times.

## Canonical Hours
1. **Prima:** Initial prayer.
2. **Tercia:** Work/Study.
3. **Sexta:** Midday prayer/Meal.
4. **Nona:** Work/Study.
5. **Vísperas:** Evening prayer.
6. **Completas:** Night rest.

## HE v3.0 Metaphor: The Cron Task
The Horarium is a system of **Policy-Driven Constraints**. If the agent is not in the Church during Prima, the "Abbot Sensor" triggers a negative reward (Obedience drop).
