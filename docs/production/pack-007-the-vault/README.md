# Pack de Producción 007: The Vault - El Oráculo del Arnés

## Overview
Este pack introduce el concepto de **The Vault** como el repositorio de verdades técnicas inmutables. Siguiendo el estándar **HE v3.0**, el Vault ha sido modularizado para garantizar la legibilidad total por parte del agente.

## Estructura Modular del Vault
- **index.md**: El catálogo maestro de la infraestructura.
- **services.md**: Contratos de microservicios y endpoints.
- **standards.md**: Diseño Lattice y estándares de identidad.
- **security.md**: Políticas de cumplimiento y seguridad monástica.
- **harness.md**: Reglas del ciclo de vida de ingeniería.

## Material Didáctico
- **Concepto:** [[the-vault-refactor]]. Por qué el Vault debe ser modular (< 200 líneas).
- **Lección de Ingeniería:** Cómo el agente consulta el Vault para pasar de una Spec a un Plan de Tareas sin alucinaciones.
- **Lore:** El Vault como el "Archivo Secreto" de la Abadía que solo Guillermo puede descifrar.
