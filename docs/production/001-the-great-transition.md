## The Hook: The End of Magic
Software development for the last few years felt like casting spells. We whispered to LLMs, and code appeared. We called it "Vibe Coding." It was fast, it was magical, and it was dangerous. Today, we are breaking the wand. In **abadIA**, we aren't asking the machine for magic; we are building an **Arnés** (Harness) to ensure engineering excellence.

## The Technology: HE v3.0
The core of this transition is **Harness Engineering (HE) v3.0**. Unlike traditional development where the human writes the code, in HE the human designs the **environment** where an agent operates. 
- **The Model** is the probabilistic brain.
- **The Harness** is the deterministic cage (Tools + Tests + Constraints).

## The Change in abadIA
We are moving from a single repository of scripts to a **Structured Genome**.
1. **Lattice Organization:** Separating atoms (functions), molecules (services), and refiners (validation logic).
2. **ADR as Law:** Every architectural choice is a contract, not a suggestion.
