# Pack de Producción 006: UX para Agentes Soberanos

## Resumen
Este pack introduce el concepto de **Agent Persona (AP)** y **Agent Journey (AJ)** como piezas fundamentales de la arquitectura de **abadIA**. No diseñamos para que el agente "esté cómodo", sino para que su exploración sea determinista y evolutiva.

## Material Didáctico
- **Concepto:** [[agent-ux-methodology]]. Por qué los User Journeys tradicionales son insuficientes para la IA.
- **Caso Práctico:** Cómo el Agent Journey de "Incursión Nocturna" utiliza el mecanismo de Ratchet para aprender de los fallos en el laberinto.
- **Lore:** Cómo la identidad de Guillermo de Baskerville (Eco) actúa como un "Arnés Moral" en el código.

## Guion para Vídeo / Short (60s)
**Escena:** Guillermo frente a la biblioteca.
**Voz Álvaro:** "¿Sabías que los User Journeys no son solo para humanos? En el proyecto abadIA hemos creado 'Agent Journeys'. No le decimos a Guillermo cómo ganar, le damos un protocolo de exploración evolutivo. Si falla, su 'Sueño Monástico' analiza el error y actualiza su contrato. Eso es Ingeniería de Arnés v3.0. Dejamos que la IA aprenda sola, pero bajo nuestras reglas soberanas. ¡Suscríbete para ver a Guillermo resolver el misterio!"
